RENDER SETTINGS DUPLICATOR

Thank you for downloading Render Settings Duplicator!

This editor plugin allows you to copy the render settings from one scene to another.

USAGE:
To open the editor window navigate to Tools -> Render Settings Duplicator

Now open up the scene from which you want to clone the Render Settings and press the copy-button on the 'Render Settings Duplicator'-window.
Then open the scene where you want the copied render settings to be applied and press the second button.
